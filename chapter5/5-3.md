java并发包提供了两个非阻塞队列的类
* ConcurrentLinkedQueue：非阻塞队列
* ConcurrentLinkedDeque：非阻塞双端队列

使用比较简单，不再贴代码，说一个使用的坑
** 不要使用size()方法 **，为什么？
因为获取size，要遍历一遍集合，看源码就知道了
```java
public int size() {
	int count = 0;
	for (Node<E> p = first(); p != null; p = succ(p))
		if (p.item != null)
			// Collection.size() spec says to max out
			if (++count == Integer.MAX_VALUE)
				break;
	return count;
}
```
